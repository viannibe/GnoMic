# GnoMic

Comparative analysis of mutational energy (ΔE) between somatic variants (COSMIC v103) and germline variants (gnomAD v4.1).

**Key result**: COSMIC somatic mutations show systematically higher ΔE than gnomAD germline across all genomic regions, substitution classes, and chromosomes.

---

## Repository structure

```
GnoMic/
├── scripts/
│   ├── 01_build_parquet.py            gnomAD CSV.gz → parquet cache
│   ├── 02_kde_region.py               ΔE by genomic region
│   ├── 03_cpg_split.py                CpG / non-CpG split
│   ├── 04_driver_passenger.py         driver / passenger stratification
│   └── 05_substitution_class.py       substitution class stratification
├── slurm/                             job submission scripts + submit_all.sh
├── notebooks/00_dataset_qc.ipynb      dataset QC and overview stats
├── results/                           output directories (generated by pipeline)
└── PREPROCESSING.md                   data sources and preprocessing steps
```

---

## Full workflow

```
RAW DATA                    PREPROCESSING (see PREPROCESSING.md)
--------                    ------------------------------------
gnomAD v4.1 VCF   →  extract fields, compute 7-mer energies, annotate VEP
COSMIC v103 VCF   →  filter SNVs, compute 7-mer energies, add driver labels
                                    ↓
                           merged_annotated.csv.gz
                           cosmic_final.tsv
                           cosmic_full_annotato.csv
                                    ↓
                            THIS PIPELINE
                            -------------
                    01_build_parquet      CSV.gz → parquet cache
                            ↓
                    02_kde_region         ΔE by genomic region
                    03_cpg_split          CpG vs non-CpG split
                    04_driver_passenger   driver / passenger stratification
                    05_substitution_class substitution class stratification
```

Steps 02–05 are independent and run in any order after step 01.

## Requirements

Input files (not included — see [PREPROCESSING.md](PREPROCESSING.md)):

| File | Used by |
|------|---------|
| `merged_annotated.csv.gz` — gnomAD v4.1 annotated with 7-mer energies | step 01 |
| `cosmic_full_annotato.csv` — COSMIC v103 with region annotation | step 02 |
| `cosmic_final.tsv` — COSMIC v103 with E scores and driver labels | steps 03–05 |

**Environment**: Python 3, conda — numpy, pandas, scipy, matplotlib, pyarrow

---

## Usage

```bash
conda activate gnomad_venv

python scripts/01_build_parquet.py          # run first
python scripts/02_kde_region.py
python scripts/03_cpg_split.py
python scripts/04_driver_passenger.py
python scripts/05_substitution_class.py     # requires parquet rebuilt with --force
```
